**ExploraAI** is an interactive Exploratory Data Analysis (EDA) Simulation & Learning Platform. Built with a sleek cyber-minimalist aesthetic, it allows users to visually and intuitively learn how data cleaning, statistical insights, and data visualizations work in the real world.

Created by **Tarannum Khan** (B.Tech AI/ML).  
[Connect on LinkedIn](https://www.linkedin.com/in/tarannum2504)

## 🚀 Features

* **01 / Data Playground**: An interactive sandbox where you can apply missing value imputation, outlier clipping (Z-score), and duplicate removal on a messy dataset, watching the distributions change in real-time.
* **02 / Data Insights**: Automated generation of correlation heatmaps, skewness analysis, and IQR-based outlier detection.
* **03 / How It Works**: A step-by-step interactive Python terminal tutorial breaking down the standard EDA workflow.
* **04 / Easy Explanation**: Complex data science concepts explained through simple everyday analogies, available in both English and Hinglish.
* **05 / Quiz Arena**: A 10-question gamified assessment module to test your knowledge with real-time scoring and streak tracking.

## 🛠️ Technology Stack
* **Frontend/Framework**: Streamlit
* **Data Processing**: Pandas, NumPy
* **Data Visualization**: Altair
* **UI/UX**: Custom HTML/CSS, Augmented-UI framework

## ⚙️ Installation & Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Tarannum2504/ExploraAI.git
   cd ExploraAI
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application:**
   ```bash
   streamlit run app.py
   ```

4. **Access the platform:**
   Open your browser and navigate to `http://localhost:8501`.
